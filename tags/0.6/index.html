Favorites Plugin

Author: Funa
Version: 0.1

Introduction:
The Favorites Plugin provides bookmarks function.

How to use:

  • Add Current Buffer: Push the plus button of toolbar or select "Add Current Buffer" of menu.
  • Delete : Push the minus button of toolbar or select "delete" of menu. You may use del key.
  • Add Path : select "Add Path" of menu.
  • Add Parent Path of Current Buffer : Push the current directory button of toolbar or select "Add Parent Path of Current Buffer" of menu.
  • Add group : Push the new directory button of toolbar or select "add group" of menu.
  • Add all buffers : Push the save all buffers button of toolbar or select "add all buffers" of menu.
